# hackathon

My Hackathon is a simple web application that allows users to add and view product reviews.

the features of this app are:
login with just username
rate products using a star system
dashboard with product insights
browse products by categories
view your personal feedback 

the works behind it:
user enters a name to create a simple account
reviews are saved using a node.js server and stored in a local json file
the app fetches product data from a free api
each review is linked to a product and displayed alongside existing reviews

how to run the project:
clone the repository
go to the backend folder
install in the terminal 'npm install'
run the server in your terminal 'node server.js'
open the index.html in your browser
